Netflix clone using React

Netflix clone website built using React.

Overview

This fully responsive Netflix clone website is built using HTML, CSS, JavaScript and React. TMDB movie API is used to access the movie data.
User autentication is done using Firebase authentication. This website is deployed and hosted using Firebase hosting.
A Youtube tutorial video was referred duing the development of the Netflix Clone website.

Movies Api

TMDB movie API is used to access the movie database. The Api gives a list of latest Netflix movies and movie trailers.

Movie Trailer

A movie trailer is played when a paricular movie is selected.
